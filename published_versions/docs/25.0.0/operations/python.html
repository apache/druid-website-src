Python Installation

Apache Druid startup script requires Python2 or Python3 interpreter. Since Python2 is deprecated, this document has instructions to install Python3 interpreter.

Python3 interpreter installation instructions

Linux

Debian or Ubuntu

- `sudo apt update`
- `sudo apt install -y python3-pip`

RHEL

- `sudo yum install -y epel-release`
- `sudo yum install -y python3-pip`

MacOS

Install with Homebrew

Refer Installing Python 3 on Mac OS X

Install the official Python release

  • Browse to the Python Downloads Page and download the latest version (3.x.x)

Verify if Python3 is installed by issuing python3 --version command.
