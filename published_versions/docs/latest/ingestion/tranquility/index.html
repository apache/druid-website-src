Tranquility

Tranquility is a separately distributed package for pushing streams to Druid in real-time.

Tranquility has not been built against a version of Druid later than Druid 0.9.2 release. It may still work with the latest Druid servers, but not all features and functionality will be available due to limitations of older Druid APIs on the Tranquility side.

For new projects that require streaming ingestion, we recommend using Druid's native support for Apache Kafka or Amazon Kinesis.

For more details, check out the Tranquility GitHub page.
